트리¶

비모수 지도학습 / 그림으로 출력하여 매우 직관적으로 이해하기 쉬운 모델¶

결정 트리 분류¶

  • 사이킷런의 DecisionTreeClassifier 사용
  • 노드에서 불순도가 가장 크게 감소하는 결정 규칙 찾음
  • 기본적으로 지니 불순도 사용

결정 트리 회귀¶

  • 사이킷런의 DecisionTreeRegressor 사용
  • 평균제곱오차(MSE)를 감소시키는지에 따라 분할

결정 트리 모델 시각화¶

  • DOT 포맷(그래프 표현 언어의 하나)로 변환 후 시각화

랜덤 포레스트 분류기¶

  • 랜덤한 결정 트리를 많이 사용하여 분류 모델 훈련
  • 사이킷런의 RandomForestClassifier 사용
  • 결정트리 : 과대적합 문제 => 앙상블 방법 : 랜덤 포레스트
  • 많은 결정 크리를 훈련, 각 트리는 부트스트랩 샘플 사용
  • 즉. 원본 샘플 수 와 동일하게 중복을 포함하여 랜덤하게 샘플 생성

랜덤 포레스트 회귀¶

  • 사이킷런 RandomForestRegressor 사용

랜덤 포레스트 중요 특성 구분, 선택¶

  • 특성 중요도 계산 및 시각화
  • 임계값 지정 -> 중요 특성 선택

부스팅 성능향상¶

  • AdaBoostClassifier, AdaBoostRegressor 사용하여 부스팅 모델 훈련
  • 이전 모델이 잘못 예측한 샘플에 높은 우선순위를 부여하는 식으로, 약한 모델을 연속적으로 훈ㄴ련
  • 최종 결과는, 더 어려운 샘플에 초점을 맞추는 약한 모델들을 모은 앙상블 모델

OOB 데이터_ 랜덤 포레스트 평가¶

  • 교차 검증을 사용하지 않고 랜덤포레스트 모델 평가
  • 랜덤포레스트_ 각 결정 트리는 부트스트랩 샘플을 사용하여 훈련 => 모든 트리는 서로 다른 일부 샘플을 훈련에 사용 X
  • 이를 OOB 데이터라함
  • OOB 샘플을 테스트 세트처럼 사용하여 랜덤 포레스트 성능을 평가
